Run the following test commands in order:

1. `npm run lint`
2. `npm run test`
3. `npm run test:acceptance`

Once all of the steps are completed, report a summary of results after all of the steps. Do not stop running all of the steps if one of them has failed.  If one of the steps has failed, show the relevant errors and diagnose the issue. Ask for permission before then applying the relevant fixes.
